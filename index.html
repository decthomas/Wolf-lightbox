Simple jQuery lightbox

Why do all lightboxes out there use a gazillion images and options when all I want is to show a larger image when I click a small one?

Features: keyboard close with Esc or X button; automatically centers in all browsers; easy to add, just add a class="lightbox" to a linked image

Browser support

Tested in IE6, IE7, IE8, IE9, Opera 10+, FF3.5+, Safari 5, Chrome 6

Files

  • reset.css includes a CSS reset; you'll probably already use one in your project.
  • modal.css includes CSS to make the modal look a bit nicer, but this CSS is not required. Ignore the file and roll your own
  • modal.js & modal.min.js: the script itself
  • buttons_close.png button image; the only image used

Usage

  • Paste the following HTML close to the end of the <body> tag (ideally this should be the last code before the end): 
    <div class="modal" style="display: none;">
	<div class="mContent">
		<a href="#" class="buttonClose" title="Close this window"><strong>Close</strong> (X)</a>
		<div class="figure">
			<img src="" alt="" />
		</div>
		<p class="caption"></p>
	</div>
</div>
<div id="modalOverlay" style="display: none;">&nbsp;</div>
  • Link a small image to a big image: <a href="image_big.jpg"><img src="image_small.jpg" alt="" /></a>
  • Add a class="lightbox" to the <a>
  • Optional: Add alt text to the <img> for the caption
